## React TODO APP
Простое SPA приложение с использованием следующих технологий:
1. ReactJS
2. Axios
3. React Router

## Запуск приложения
Используйте `npm start` что бы запустить приложение в режиме разработчика

## Production 
Используйте `npm build` что бы "скомпилировать" приложение в папку `build`
